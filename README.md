# asciilator

A lightweight Rust library for converting images to ASCII art.
Designed for the **dutar** music player to render album covers in the terminal.

## Quick Start

```rust
use ascillator::{convert_to_ascii, AsciilatorConfig};
use image::open;

fn main() {
    // 1. Load image
    let img = open("cover.jpg").expect("File not found");

    // 2. Configure (optional, defaults work for most cases)
    let config = AsciilatorConfig {
        width: 60,          // Output width
        scale_factor: 0.55, // Adjusts for non-square terminal fonts
        ..Default::default()
    };

    // 3. Print
    println!("{}", convert_to_ascii(&img, &config));
}
```

## Configuration

| Field | Default | Description |
| --- | --- | --- |
| `width` | `100` | Width of the output in characters. |
| `scale_factor` | `0.55` | Correction for tall terminal fonts. Use `0.5` - `0.6` to fix aspect ratio. |
